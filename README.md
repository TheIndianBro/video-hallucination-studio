# Video Hallucination Studio

A dedicated studio environment for analyzing, debugging, and correcting hallucinations in AI-generated videos. This project provides a workspace to visualize detection results and experiment with prompt refinements for video generation models.

## Features

- **Video Analysis Workspace**: A specialized interface for reviewing AI-generated videos frame-by-frame or scene-by-scene.
- **Hallucination Highlighting**: Visual indicators for detected hallucinations within the video timeline.
- **Correction Studio**: Tools to modify prompts and parameters to regenerate specific segments or the entire video.
- **Comparison View**: Side-by-side comparison of original and corrected videos.

## Tech Stack

- **Framework**: React, Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/TheIndianBro/video-hallucination-studio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd video-hallucination-studio
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Usage

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser and navigate to the local server address (usually `http://localhost:5173`).
